--- layout: main title: "RoboZombie | Setup" ---
If you opt to use the out-of-the-box JSON (de)serializer add the Gson dependency; likewise add the Simple-XML dependency for the XML (de)serializer. Or simply download the Gson and Simple-XML JARs and add them to your libs folder.

1. For Maven based projects.


<dependency>
    <groupId>com.lonepulse</groupId>
    <artifactId>robozombie</artifactId>
    <version>1.3.2</version>
</dependency>

2. For Gradle based projects.


repositories {
    mavenCentral()
}

dependencies {
    compile 'com.lonepulse:robozombie:1.3.2'
}
