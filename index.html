RoboZombie

/rō-bō'zŏm'bē/ noun. 1 A lightweight HTTP facade which simplifies network communication. 2 An endpoint proxy generator for RESTful web services.

"Effortless networking for your Android application"

Mirror an endpoint API by annotating an interface and inject the generated proxy.


Usage

Here's your Model

public class Conversion implements Serializable {

        private String lhs;
        private String rhs;
        private String error;
        private boolean icc;

        /* serialVersionUID, Accessors, Mutators, 
         * hashCode(), equals() and toString() omitted */
}


Define the Endpoint Interface

@Endpoint("www.google.com")
@Parser(Parser.PARSER_TYPE.JSON)
public interface ConverterEndpoint {

    @Request(path = "/ig/calculator")
    Conversion convert(@Param("q") String query);
}


Inject and Invoke

@Bite
private ConverterEndpoint endpoint;
{
    Zombie.infect(this);
}

...

Conversion rate = endpoint.convert("1USD=?AUD");


For more information refer the complete usage guide on the project wiki.


Setup

For maven based Android projects add the following dependency in your pom.xml.

<dependency>
   <groupId>com.lonepulse</groupId>
   <artifactId>robozombie</artifactId>
   <version>1.2.3</version>
   <type>apklib</type>
</dependency>


For standard Android projects download the repository (tarball, zipfile) or clone it and import it as an existing project in Eclipse.

$ git clone git://github.com/sahan/RoboZombie.git
